# QRONEX'S STUDIOS AI


## 🛠️ Installation
```bash
# Clone the repository
git clone 

# Navigate to project directory
cd qronex_ai

# Install dependencies
pnpm install

# Start development server
pnpm run dev
```

## 🔧 Development
```bash
# Build for production
pnpm run build

# Preview production build
pnpm run preview
```
